Usage

In this tutorial we'll see how to setup your first Roblox bot using boblox.ts!

Getting your Roblox cookieâ€‹

In order to login to Roblox you'll need a cookie. Getting this cookie is very simple :

  • Go to roblox.com
  • Login if you're not logged in
  • Press F12
  • Go to Application Tab then click cookies
  • Copy the value of the cookie named .ROBLOSECURITY
Do not share your cookie with anyone!

Sharing your cookie will allow anyone to log in your account and steal your robux and items.

Javascript Exampleâ€‹

const boblox = require('boblox.ts');

const client = boblox.Client();

client.on('ready', () => {
  console.log('We are logged in as ' + client.user.name);
});
client.login('Your roblox cookie');

Typescript Exampleâ€‹

import boblox from 'boblox.ts';

const client = boblox.Client();

client.on('ready', () => {
  console.log('We are logged in as ' + client.user.name)
});
client.login('Your roblox cookie')

Hiding your cookieâ€‹

You should not put your roblox cookie directly in your source code. The best way to hide it is by using dotenv

  1. Create a file named .env along with your source code
  2. Write inside COOKIE=your cookie here
  3. Install dotenv by using npm install dotenv or yarn add dotenv

Typescriptâ€‹

import {config} from 'dotenv';
config();
// COOKIE is now accessible in process.env
client.login(process.env.COOKIE); // You can login like this

Javascriptâ€‹

const dotenv = require('dotenv');
dotenv.config();
// COOKIE is now accessible in process.env
client.login(process.env.COOKIE); // You can login like this

Nice you are now using boblox.ts!
